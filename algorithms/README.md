## Algorithms

Selected algorithm implementations from coursework, demonstrating
problem-solving skills and fundamental algorithmic concepts relevant
to computational biology and data analysis.

The focus of this work is on translating biological questions into
step-by-step computational solutions.

### Topics Covered
- Algorithmic problem decomposition
- Sequence parsing and manipulation
- Codon-based translation and ORF detection
- Consensus sequence construction
- Comparative sequence analysis
